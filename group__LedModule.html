int  dmaChannel,
size_t  maxLeds 
)

Initialize the LED library.

Must only be called once at program startup and must be called before led_setColors or led_finalize.

Parameters
[in]dataPinThe LED string's GPIO data pin.
[in]dmaChannelThe DMA channel to use for communication.
[in]maxLedsThe maximum number of LEDs expected.
Returns
True if successful, false otherwise.
