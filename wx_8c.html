Convert a node's text as an ISO-8601 date/time string.

Assumes UTC and ignores timezone information. Assumes integer seconds. Performs basic sanity checks, but does not check if the day exceeds the number of days in the specified month.

The output date/time will be zeroed if the date/time string is invalid.

static char * trimLocalId ( const char *  id,
size_t  maxLen 
)
static

Trims a local airport ID for display.

Non-ICAO airport IDs include numbers and are three characters long, e.g. 7S3 or X01. However, AviationWeather.gov expects four-character IDs. So, for the US, 7S3 should be "K7S3". If the specified ID has a number in it, return a duplicate string that does not have the K. If the ID is an ICAO ID, return a duplicate of the original string.

Parameters
[in]idThe airport ID of interest.
[in]maxLenThe maximum number of characters permitted.
Returns
A duplicate of either the original ID or the shortened non-ICAO ID.
