Configure a 3D shader.

Sets up the position, color, and texture coordinate attribute arrays assuming an array of Vertex3D structs will be used. Binds the specified textures sequentially starting from texture unit 0. Sets the view matrix to the specified transform and the projection matrix to the screen projection.

Parameters
[in]rsrcThe gfx context.
[in]programThe shader program to use.
[in]viewThe view transform to use.
[in]modelThe model transform to use.
[in]texturesAn array of textures to bind.
[in]textureCountThe number of textures to bind.
