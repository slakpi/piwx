Load the PiWx configuration.

Loads the PiWx configuration from configFile or initializes the configuration structure with the PiWx defaults if the file does not exist or is invalid.

Parameters
[in]installPrefixThe PiWx installation prefix path.
[in]imageResourcesThe path to PiWx's image resources.
[in]fontResourcesThe path to PiWx's font resources.
[in]configFileThe path to PiWx's configuration file.
Returns
An initialized configuration structure or NULL if unable to allocate memory for the structure or any of the paths is invalid.
