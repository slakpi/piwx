A POSIX-only function to append a trailing backslash to prefix, if necessary, followed by name.

Parameters

[in]lenLength of path.
[in]prefixNull-terminated path prefix.
[in]fileNull-terminated file name.

The buffer pointer or NULL if the output buffer is too small.
