◆ gfx_beginLayer()

void gfx_beginLayer ( DrawResources  resources,
Layer  layer 
)

Begin drawing to the specified cache layer.

If necessary, creates the framebuffer and texture objects to receive the draw commands. Draw commands must be followed by gfx_endLayer. The cached texture can be drawn with gfx_drawLayer.

Parameters
[in]resourcesThe gfx context.
[in]layerThe layer cache.
